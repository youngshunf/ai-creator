## 1. Logo 规范

### 1.1 Logo 版本

| 版本 | 文件 | 尺寸 | 用途 |
|------|------|------|------|
| **主 Logo** | `logo-refined.svg` | 200×200px | 品牌主视觉、大尺寸展示 |
| **白底版** | `logo-white.svg` | 200×200px | 深色背景使用 |
| **图标版** | `logo-icon.svg` | 64×64px | Favicon、App 图标、小尺寸 |
| **横版** | `logo-horizontal.svg` | 280×64px | 网站头部、文档、名片 |

### 1.2 Logo 构成元素

```yaml
设计元素:
  流动曲线:
    - 主曲线: 10px 宽度，代表内容创作的流畅过程
    - 辅助曲线: 4px 宽度，增加层次感
    - 方向: 从左下到右上，象征向上发展

  笔触元素:
    - 笔尖: 菱形形状，代表创作起点
    - 笔杆: 6px 宽度线条，象征创意诞生
    - 笔迹: 5px 宽度，连接到流动曲线

  灵感粒子:
    - 主光点: 10px 半径，带发光效果
    - 次级光点: 3-5px 半径，不同透明度
    - 分布: 右上角区域，暗示内容分发
```

### 1.3 Logo 安全区域

```
┌──────────────────────────────────────┐
│                                      │
│    ┌─────────────────────────┐      │
│    │                         │      │
│    │         LOGO            │      │ ← 安全区域 = Logo 高度的 25%
│    │                         │      │
│    └─────────────────────────┘      │
│                                      │
└──────────────────────────────────────┘
```

### 1.4 Logo 最小尺寸

| 版本 | 最小尺寸 | 说明 |
|------|---------|------|
| 主 Logo | 48×48px | 保持细节可识别 |
| 图标版 | 16×16px | Favicon 最小尺寸 |
| 横版 | 140×32px | 保持文字可读 |

### 1.5 Logo 使用禁忌

```yaml
禁止:
  - 拉伸或压缩 Logo
  - 更改 Logo 颜色
  - 添加阴影或特效
  - 在复杂背景上使用
  - 旋转 Logo
  - 分离 Logo 元素单独使用
  - 修改元素比例
```

### 2.4 渐变色

```css
/* 主渐变 - 品牌标识 */
--gradient-primary: linear-gradient(135deg, #3B82F6 0%, #8B5CF6 100%);

/* 三色渐变 - Logo 背景 */
--gradient-logo: linear-gradient(135deg, #3B82F6 0%, #6366F1 50%, #8B5CF6 100%);

/* Hero 背景渐变 - 明亮版 */
--gradient-hero: linear-gradient(135deg, #4338CA 0%, #6366F1 50%, #3B82F6 100%);

/* Hero 背景渐变 - 浅色版 (适用于浅色模式) */
--gradient-hero-light: linear-gradient(135deg, #EEF2FF 0%, #E0E7FF 50%, #DBEAFE 100%);

/* 浅色渐变 - 卡片背景 */
--gradient-light: linear-gradient(135deg, rgba(59, 130, 246, 0.08) 0%, rgba(139, 92, 246, 0.08) 100%);

/* 强调渐变 - 卡片悬浮 */
--gradient-accent: linear-gradient(135deg, rgba(59, 130, 246, 0.15) 0%, rgba(139, 92, 246, 0.15) 100%);

/* 文字渐变 */
--gradient-text: linear-gradient(135deg, #60A5FA 0%, #A78BFA 100%);

/* 明亮文字渐变 */
--gradient-text-bright: linear-gradient(135deg, #3B82F6 0%, #8B5CF6 100%);
```

### 2.5 色彩可访问性

```yaml
对比度要求 (WCAG 2.1):
  正文文字: 最小 4.5:1
  大标题: 最小 3:1
  交互元素: 最小 3:1

推荐组合:
  - #1F2937 on #FFFFFF: 12.6:1 ✓
  - #6B7280 on #FFFFFF: 5.0:1 ✓
  - #3B82F6 on #FFFFFF: 3.9:1 ✓ (大文字/图标)
  - #FFFFFF on #3B82F6: 3.9:1 ✓ (按钮文字)
  - #1F2937 on #EEF2FF: 11.2:1 ✓ (浅色背景)
  - #3B82F6 on #EEF2FF: 3.5:1 ✓ (浅色背景链接)
```

## 3. 字体规范

### 3.1 字体族

```css
/* 中文字体 */
font-family: 'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', sans-serif;

```

### 3.2 字体大小

| 级别 | 大小 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| **H1** | 3.5rem (56px) | 1.2 | 800 | 首屏大标题 |
| **H2** | 2.5rem (40px) | 1.3 | 800 | 章节标题 |
| **H3** | 1.5rem (24px) | 1.4 | 700 | 卡片标题 |
| **H4** | 1.2rem (19px) | 1.5 | 700 | 小标题 |

| **Body Small** | 0.9rem (14px) | 1.5 | 400 | 辅助说明 |

### 3.3 字重

```yaml
字重映射:
  400: Regular - 正文
```

## 9. 卡片规范

### 9.1 卡片类型

```yaml
基础卡片:
  背景: white
  圆角: 20px
  边框: 1px solid var(--light-gray)
  内边距: 2.5rem
  阴影: none (默认)
  悬浮阴影: var(--shadow-lg)
  悬浮变换: translateY(-10px)

特色卡片:
  背景: var(--gradient-primary)
  圆角: 24px
  内边距: 3rem
  文字: white

玻璃卡片 (Glassmorphism):
  背景: rgba(255, 255, 255, 0.1)
  模糊: backdrop-filter: blur(10px)
  边框: 1px solid rgba(255, 255, 255, 0.2)
  圆角: 20-24px
```

### 9.2 容器样式原则

```yaml
核心原则:
  - 避免使用 border 边框
  - 用浅色背景或浅色阴影区分不同容器
  - 保持页面干净整洁

区分策略:
  色差较大时:
    - 容器背景与页面背景色差明显
    - 不需要边框和阴影
    - 仅靠背景色区分即可

  色差接近时:
    - 容器背景与页面背景色差较小
    - 使用浅阴影 (shadow-sm 或 shadow-xs)
    - 避免使用边框

推荐做法:
  ✓ 白色卡片 + 浅灰背景 (无边框)
  ✓ 浅色背景 + 微阴影
  ✓ 渐变背景容器 (无边框无阴影)

避免做法:
  ✗ 1px solid border 边框
  ✗ 边框 + 阴影同时使用
  ✗ 深色边框
```

## 10. 动效规范

### 10.1 过渡时间

```yaml
过渡时间:
  fast: 0.15s   - 按钮状态、图标
  normal: 0.3s  - 卡片悬浮、导航
  slow: 0.5s    - 页面切换、大型动画

缓动函数:
  default: ease
  smooth: ease-in-out
  bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55)
```

### 10.2 常用动效

```css
/* 卡片悬浮 */
.card:hover {
    transform: translateY(-10px);
    box-shadow: 0 30px 60px rgba(0, 0, 0, 0.1);
}

/* 按钮悬浮 */
.btn-primary:hover {
    transform: translateY(-3px);
    box-shadow: 0 20px 40px rgba(59, 130, 246, 0.4);
}

/* 链接悬浮 */
.link:hover {
    color: var(--primary-blue);
}

/* 图标缩放 */
.icon:hover {
    transform: scale(1.1);
}

/* 光标闪烁 */
@keyframes blink {
    0%, 50% { opacity: 1; }
    51%, 100% { opacity: 0; }
}
```

## 11. 响应式断点

### 11.1 断点定义

| 断点 | 宽度 | 设备 |
|------|------|------|
| **xs** | < 480px | 小手机 |
| **sm** | 480px - 767px | 大手机 |
| **md** | 768px - 1023px | 平板 |
| **lg** | 1024px - 1279px | 小桌面 |
| **xl** | >= 1280px | 大桌面 |

### 11.2 响应式规则

```yaml
桌面端 (>= 1024px):
  - 完整导航
  - 多列布局
  - 大尺寸标题

平板端 (768px - 1023px):
  - 简化导航
  - 2列布局
  - 中等尺寸标题

移动端 (< 768px):
  - 汉堡菜单
  - 单列布局
  - 紧凑标题
  - 堆叠按钮
```
